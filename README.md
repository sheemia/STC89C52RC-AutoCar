## 📖 项目背景
[cite_start]根据竞赛题目要求 [cite: 8, 9, 10]，小车需要在 220cm×120cm 的场地上完成以下任务：
1.  **基础任务**：直线行驶、弧线循迹 (A->D)、定点停车声光提示。
2.  **复杂路径**：多点巡航 (A->B->C->D->A)，包含直角转弯和半圆弧路径。
3.  **盲走任务**：在无黑线引导区域 (C->B)，利用陀螺仪保持直线并精准切入下一段赛道。

**本项目成功完成了所有基础及发挥部分任务，且时间均优于题目限制 (50s)。**

## 🛠️ 硬件架构

| 模块 | 型号/参数 | 选型理由 |
| :--- | :--- | :--- |
| **MCU** | STC89C52RC | [cite_start]比赛指定芯片，资源虽少但经典稳定 [cite: 25] |
| **电机驱动** | TB6612FNG | MOSFET架构，效率高，不发热，体积小 (优于L298N) |
| **电机** | MG310 直流减速电机 | 强磁碳刷，带编码器/减速箱，扭矩大 |
| **循迹传感器** | 感为 8路灰度传感器 | 相比2路/4路分辨率更高，能识别特殊路口 |
| **姿态传感器** | JY61P (MPU6050) | 串口输出 Yaw 角，用于无引线区的直线保持 |
| **显示交互** | 0.96寸 OLED (I2C) | 实时显示任务状态、圈数、传感器数值 |
| **电源** | 2x18650 锂电池 (7.4V) | 动力足，配合LM2596降压给单片机供电 |

## 💻 核心算法与软件亮点

### 1. 位置式 PID 闭环控制
摒弃传统的 `if-else` "摇头"循迹，采用位置式 PD 算法，使小车在直线和弯道均能平滑行驶。
* **P (比例)**: 快速响应黑线偏差。
* **D (微分)**: 预测误差趋势，抑制过冲和震荡。
* **参数**: 经过实测，`Kp=5, Kd=25` (在 `Motor.c` 中定义)。

### 2. 陀螺仪姿态融合 (Gyro Fusion)
在任务5和任务7中，针对无黑线区域：
* **模式切换**: 状态机检测到 "全白" 信号后，自动从灰度循迹模式切换至陀螺仪模式。
* **相对角度锁定**: 调用 `Reset_Yaw_Reference()` 锁定当前航向，计算 `Target_Yaw - Current_Yaw` 作为误差输入 PID，实现 "盲走直线"。

### 3. 鲁棒性设计
* **防跑飞逻辑**: 当传感器丢失信号时，保持上一时刻的偏差值，防止小车原地打转。
* **路口滤波**: 引入 `Forward_ABit()` 逻辑，配合位掩码 (Mask) 运算，防止在十字路口重复计数。
